# PWA 图标说明

## 当前状态
本目录包含临时的 SVG 图标，使用 📝 emoji 作为占位符。

## 生成真实 PNG 图标
建议使用以下工具之一生成真实的 PNG 图标：

### 方法1

替换`generate-svg-to-png.html`中的svg，浏览器打开，下载图标。

### 方法2：在线工具
1. 访问 https://favicon.io/emoji-favicons/memo/
2. 下载生成的图标包
3. 替换本目录中的 SVG 文件

### 方法3：使用设计工具
创建以下尺寸的 PNG 图标：
- icon-192.png (192x192)
- icon-512.png (512x512)
- icon-maskable-512.png (512x512, 带安全区)
- apple-touch-icon.png (180x180)



## 设计建议
- 背景色：#3b82f6 (蓝色)
- 图标：简洁的笔记/博客相关符号
- Maskable图标：确保核心内容在中心圆形安全区内（直径80%）

## 文件清单
- icon-192.svg → 替换为 icon-192.png
- icon-512.svg → 替换为 icon-512.png
- icon-maskable-512.svg → 替换为 icon-maskable-512.png
- apple-touch-icon.svg → 替换为 apple-touch-icon.png
